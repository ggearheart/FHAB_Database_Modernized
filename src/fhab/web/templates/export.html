Provisional API

Read-only and CORS-open, so apps like the CyanoSafe demo can consume it directly. The index lists all datasets:

/api/open/index.json

It reflects the current database (including reports not yet in an official data.ca.gov release), so every record is marked provisional. Responses are cached for 10 minutes.

Scheduled generation

For a daily/weekly batch, run python scripts/export_flatfiles.py OUTPUT_DIR from cron or a Render cron job. See docs/OPEN_DATA.md.
